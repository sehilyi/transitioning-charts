# Transitioning Charts
This repo creates a simple HTML page which is used in a study on introducing new genomics chart types to a blind individual.
> [!NOTE]
> This repo is a work in progress.

## How to run
To run:
```sh
pnpm install
pnpm run dev
```

To deploy:
```sh
pnpm run predeploy
pnpm run deploy
```
